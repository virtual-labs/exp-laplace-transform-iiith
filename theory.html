Laplace transform

The Laplace transform of a continuous time signal x(t)\text{x}(t)x(t) can be expressed as: X(s)=∫−∞∞x(t)e−stdt\text{X}(s) = \int_{-\infty}^{\infty} \text{x}(t) e^{-st} dtX(s)=∫−∞∞​x(t)e−stdt

Where s=σ+jωs = \sigma + j \omegas=σ+jω represent a complex frequency on a complex number plane.

e−st=e−(σ+jω)t=e−σte−jωt e^{-st} = e^{-(\sigma+j\omega)t} = e^{-\sigma t} e^{-j\omega t} e−st=e−(σ+jω)t=e−σte−jωt

We can clearly see that

Re{e−st}=e−σtcos⁡(ωt)Re\{e^{-s t}\} = e^{-\sigma t} \cos(\omega t) Re{e−st}=e−σtcos(ωt)

Im{e−st}=−e−σtsin⁡(ωt)Im\{e^{-s t}\} = - e^{-\sigma t} \sin(\omega t)Im{e−st}=−e−σtsin(ωt)

Example - All RLC circuits are LTI systems. Recall that for any AC source, we always get sinusoidal voltage/current across any part of the circuit. The amplitude and phase might change but the shape remains the same

The Laplace transform has always two parts:

  1. Mathematical expression

  2. Region of convergence: the region in the complex s-plane, where the mathematical expression is valid.

Example

  1. x(t)=e−tu(t)\text{x}(t) = e^{-t} u(t) x(t)=e−tu(t)

    Solution: The Laplace transform is given as:

Linear constant coefficient differential equations (LCCDE)

Any linear constant coefficient differential equations (LCCDE) in Laplace domain can be expressed as ratio of polynomials.

Example: Let x(t)=dydt+a y(t)\text{x}(t) = \frac{d\text{y}}{dt} + a~\text{y}(t)x(t)=dtdy​+a y(t)

Taking Laplace transform of the above equation:

           X(s)=sY(s)+aY(s) ~~~~~~~~~~~\text{X}(s) = s\text{Y}(s) + a \text{Y}(s)           X(s)=sY(s)+aY(s)

           X(s)=(s+a) Y(s)~~~~~~~~~~~ \text{X}(s) = (s + a)~ \text{Y}(s)           X(s)=(s+a) Y(s)

           Y(s)=X(s)s+a~~~~~~~~~~~\text{Y}(s) = \frac{\text{X}(s)}{s + a}           Y(s)=s+aX(s)​

           H(s)=Y(s)X(s)=1s+a~~~~~~~~~~~ \text{H}(s) = \frac{\text{Y}(s)}{\text{X}(s)} = \frac{1}{s + a}           H(s)=X(s)Y(s)​=s+a1​

Here H(s)\text{H}(s)H(s) is known as transfer function or system function.

Application of Laplace transform for system analysis

Let h(t)h(t)h(t) be the system with input x(t)\text{x}(t)x(t) and output being y(t)\text{y}(t)y(t).

     y(t)=x(t)∗h(t)~~~~~\text{y}(t) = \text{x}(t) * h(t)     y(t)=x(t)∗h(t)

The system input-output relationship in Laplace transform can be expressed as:

   Y(s)=X(s)H(s)~~~\text{Y}(s) = \text{X}(s) \text{H}(s)   Y(s)=X(s)H(s)

We are interested in additional system properties and their effect on impulse response and system function

Causality of LTI system

For LTI system to be causal, we should have

    h(t)=0,  ∀ t<0~~~~h(t) =0, ~~\forall~ t<0     h(t)=0,  ∀ t<0

    y(t)=x(t)∗h(t)=∫−∞∞x(τ)h(t−τ)dτ~~~~ \text{y}(t) = \text{x}(t)*h(t) = \int_{-\infty}^{\infty} \text{x}(\tau) h(t-\tau) d\tau    y(t)=x(t)∗h(t)=∫−∞∞​x(τ)h(t−τ)dτ

For Causality: we should only use x(τ)\text{x}(\tau)x(τ) for τ≤t\tau \leq tτ≤t

τ>t, h(t−τ)=0\tau > t,~ h(t-\tau) = 0 τ>t, h(t−τ)=0

h(t)=0  ∀ t<0h(t) = 0~~ \forall~ t<0h(t)=0  ∀ t<0

  • For a causal system (i.e.,h(t)h(t)h(t) is a right-sided signal) , what is nature of H(s)\text{H}(s)H(s) ?

  • ROC of H(s)\text{H}(s)H(s) will be right-sided plane. (Converse is not true in general)

  • For a system with rational system function, causality is equivalent to the ROC being right-sided plane to the right of right most pole.

          h(t)=e−tu(t)~~~~~~~~~~h(t) = e^{-t} u(t)          h(t)=e−tu(t) (Causal)

  1. H(s)=ess+1\text{H}(s) = \frac{e^s}{s+1}H(s)=s+1es​ and Re{s}>−1Re\{s\} > -1Re{s}>−1

Stability of LTI system (bounded input bounded output)

An LTI system is said to be stable if

∫∞∞∣h(t)∣dt<∞\int_{\infty}^{\infty} |h(t)| dt < \infty∫∞∞​∣h(t)∣dt<∞

i.e. h(t)h(t)h(t) is absolutely integrable

We know Laplace transform is given as:

    H(s)=∫∞∞h(t)e−stdt~~~~\text{H}(s) = \int_{\infty}^{\infty} h(t) e^{-st} dt    H(s)=∫∞∞​h(t)e−stdt

H(s)∣s=0=∫∞∞h(t)dt<∞ \text{H}(s)|_{s=0} = \int_{\infty}^{\infty} h(t) dt < \inftyH(s)∣s=0​=∫∞∞​h(t)dt<∞

Laplace transform converges at s=0s=0s=0, In fact, for any s=jω=0+jω, H(s)∣_s=jω<∞→s=j\omega = 0+j\omega, ~\text{H}(s)|\_{s=j\omega} < \infty \rightarrow s=jω=0+jω, H(s)∣_s=jω<∞→ system is stable

Note: jωj\omegajω-axis i.e. Re{s}=0Re\{s\} = 0Re{s}=0 is part of the ROC

Example:

  1. Shifting operator: h(t)=δ(t−t0)h(t) = \delta(t-t_0)h(t)=δ(t−t0​)

    Taking Laplace transform: H(s)=est0 \text{H}(s) = e^{st_0}H(s)=est0​
